Golden Template

This app is the canonical Flask starter for all projects in this portfolio. It demonstrates the patterns documented in PATTERNS.md — copy the app/auth/ blueprint, db.py, and config.py into any new project to get auth and database support immediately.

Technology Stack
  • Backend: Flask 3.1, Python 3.10+
  • Auth: Google OAuth via Authlib
  • Database: SQLite (dev) / Postgres (prod) via psycopg3
  • Frontend: Bootstrap 5.3, Jinja2
  • Server: Gunicorn + Nginx + systemd
  • Deploy: flask-deploy
